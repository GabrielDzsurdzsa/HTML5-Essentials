article

The article tag is a container type tag and can contain a: blog post, news story, forum post, comment etc.

It is typically made up of the following HTML tags:

  • h1 - head title
  • span - inline text
  • p - paragraph
  • button - button

Example:

<article>
  <h1>Title</h1>
  <p>Article paragraph text.</p>
</article>

aside

The aside tag is a container type tag and can contain: contact info, social and share icons, a profile badge etc.

It is typically made up of the following HTML tags:

  • h1 - head title
  • p - paragraph
  • button - button

Example:

<aside>
  <h1>Title</h1>
  <p>Contact phone number.</p>
  <p>Contact address.</p>
  <p>Social Links.</p>
</aside>

canvas

The canvas tag is used to draw graphics on a page via Javascript. It is also a container type but does not take HTML tags directly. Rather it renders HTML through javascript.

Example

<canvas id="myCanvas" width="200" height="50" >
</canvas>

figure and figcaption

The figure tag is a container type tag and can contain: image, image caption, SVG graphics, canvas graphics.

It is typically made up of the following HTML tags:

  • img - image
  • canvas - canvas graphics
  • svg - SVG graphics
  • figcaption - caption

Example:

<figure>
  <img src=""></img>
  <figcaption>Image caption.</figcaption>
</figure>

footer

The footer tag is a container type tag and can contain: reference links, subscription links, share & social media buttons, a navigation menu, copyright & other information etc.

It is typically made up of the following HTML tags:

  • a - link
  • ul - list
  • button - button
  • span - inline text

Example:

<footer>
  <span>Meta.</span>
  <span>Subscription link.</span>
  <span>Social Links.</span>
</footer>

header

The header tag is a container type tag and can contain: a logo, a website title, share & social media icons, search, navigation etc.

It is typically made up of the following HTML tags:

  • img - logo image
  • h1 - website title
  • button - button
  • input - text input for search
  • nav - navigation

Example:

<header>
  <h1>
    <figure>
      <img src="" alt="Site Logo" /> Site Title
    </figure>
  </h1>
  <nav>
    <a href="#">Link 1</a>
    <a href="#">Link 2</a>
    <a href="#">Link 3</a>
  </nav>
</header>

main

The main tag is a container type tag and can contain: all html content that is not of a header, footer and navigation type.

It is typically made up of the following HTML tags:

  • section - section
  • div - division
  • button - button
  • ul - un-ordered list
  • aside - aside
  • mark - highlight text

Example:

<main>
  <section>
    <article>Article contents.</article>
  </section>
</main>

meter

The meter tag can contain: a metered gauge that shows either a percentage or number and would normally render as follows:

  • 60% - Disk usage
  • 2 out of 10 - Query results

Example:

<meter value="0.6">
</meter>
<meter value="2" min="0" max="10">
</meter>

The nav tag is a container type tag that can contain: a navigation menu.

It is typically made up of the following HTML tags:

  • ul - list

progress

The progress tag can contain: a progress bar and would normally render as follows:

section

The section tag is a container type tag and can contain: articles, images, videos, menus, other media etc.

It is typically made up of the following HTML tags:

  • article - article
  • aside - aside
  • div - division
  • img - image

svg

svg or Scalable Vector Graphics is a w3c recommended open source standard, a container type tag and can contain: any svg style html element e.g. rect

It is typically made up of the following HTML tags:

  • rect - rectangle
  • circle - circle
  • line - line
  • polyline - polyline
  • path - line path

Example:

<svg width="100" height="100">
  <rect x="50" y="50" width="150px" height="50px" stroke="lightBlue" stroke-width="10" fill="lightGray" ><rect/>
</svg>

video

Example:

  <source src="mov_bbb.ogg"><source/>
</video>

What is HTML5 Essentials?

HTML5 Essentials is a form based HTML document builder.

Completely client side, compact and responsive, HTML5 Essentials is designed for the beginner web developer.

It can also be used as a teaching tool demonstrating the latest HTML technologies.

The HTML Document

HTML5 Essentials creates complete HTML markup that can be saved as a hostable .html document.

View and download your .html document from the document menu.

HTML5 Tags

HTML5 Essentials simplifies adding HTML5 tags to an HTML document by populating tag proprieties and then rendering the tag in a container.

There are many HTML and HTML5 tags to consider. HTML5 Essentials focuses only on the most essential, up to date and relevant ones.

How can I add a new tag too my HTML Document?

  1. Click on + New button belonging to desired HTML tag
  2. Fill out the form fields
  3. Click on Add to HTML in the window
  4. Go to your document menu.

How can I start a new HTML document?

You can just close the browser window and re-launch the app.

Privacy

We do not store or share your information. HTML5 Essentials is a client-side only, standalone application. It does not connect to any databases for storage.

All data is stored in your local browser session only, no cookies.
